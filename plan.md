## P0 — 核心体验提升

### 1. 记忆衰减与巩固机制
- [x] 记忆重要性衰减：长期不被访问的记忆 importance 逐渐降低
- [x] 记忆巩固：定期用 LLM 将多条相关记忆合并/压缩
- [x] 休眠记忆唤醒：超过 30 天没被召回但语义相关的记忆主动浮现
- 相关文件：`backend/app/core/memory.py`, `backend/app/db/vector.py`

### 2. 用户偏好自动学习
- [x] 从对话中自动学习回复长度偏好
- [x] 话题兴趣权重学习
- [x] 情感表达偏好学习（温柔 vs 毒舌的接受度）
- [x] 偏好带置信度分数，随对话次数递增
- [x] 偏好影响人格引擎的行为参数
- 相关文件：新增 `backend/app/core/preferences.py`

### 3. 情绪轨迹分析与预警
- [ ] 7/14/30 天情绪趋势可视化（valence/arousal 走向）
- [ ] 5 种预警模式：情绪恶化、持续低落、社交孤立、愤怒升级、自我伤害念头
- [ ] 学习哪些话题让用户情绪好转/恶化
- [ ] 预警时调整语晴的主动关心行为
- 相关文件：`backend/app/core/emotion.py`, 新增前端图表组件

## P1 — 认知深度增强

### 4. 矛盾与认知扭曲检测
- [ ] 对比用户当前陈述与过去 30 天记忆，发现自我认知矛盾
- [ ] 检测 5 种 CBT 认知扭曲（全有全无、过度概括、自我贬低、忽视积极面、情绪化推理）
- [ ] 以语晴的口吻委婉指出，不是说教
- 相关文件：新增 `backend/app/core/contradiction.py`

### 5. 目标追踪
- [ ] 从对话中检测用户目标（工作、学习、健康、关系等）
- [ ] 追踪进度，后续对话中自然跟进
- [ ] 目标停滞时给出温和推动
- 相关文件：新增 `backend/app/core/goals.py`

### 6. 元记忆系统
- [ ] 追踪"用户的想法如何演变"，不仅是记住什么
- [ ] 识别想法的催化剂（挫折、灵感、限制、机会）
- [ ] 追踪想法经历阶段（细化→扩展→修正→综合→突破）
- 相关文件：`backend/app/core/memory.py` 扩展
